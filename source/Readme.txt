Iron Framework

--------------------------------------------
-- Building Iron Framework
--------------------------------------------
You need download and install following components:

 Entity framework 4.3 RTM
 Enterprise Library 5.0  RTM 
 Newtonsoft.Json 
 Moq 4 
 ASP.NET MVC 3 RTM Tools Update
 WCF Web API Preview 4
 AutoMapper.2.0.0
 ASP.NET MVC 4 RTM for Visual Studio 2010 SP1

--------------------------------------------
-- DataBase
--------------------------------------------
 We use the SQL SERVER Sample database which named AdventureWorks. 
 Please install it before you set-up environment

 Install AdventureWorks database of SQL Server 2005 SP2 
 http://msftdbprodsamples.codeplex.com/releases/view/4004
